# 數碼寶貝 RPG 格鬥遊戲 🎮

一款精美的回合制數碼寶貝格鬥遊戲，使用純 HTML、CSS 和 JavaScript 開發。

## ✨ 遊戲特色

- 🎨 **精美的視覺設計** - 漸變色彩、玻璃態效果、流暢動畫
- ⚔️ **回合制戰鬥系統** - 策略性的技能選擇和 MP 管理
- 🔥 **屬性相剋系統** - 火焰、水系、自然、光明、黑暗五大屬性
- 🤖 **智能 AI 對手** - 具有策略性的敵方 AI
- 📖 **數碼圖鑑** - 收錄 12 隻經典數碼寶貝
- 🎯 **技能系統** - 每隻數碼寶貝擁有獨特的技能組合

## 🎮 如何遊玩

### 方法 1: 直接在瀏覽器中打開

1. 在檔案總管中找到 `c:\digimon\index.html`
2. 雙擊檔案，會在預設瀏覽器中打開
3. 開始遊戲！

### 方法 2: 使用本地伺服器（推薦）

如果你安裝了 Python，可以使用以下命令啟動本地伺服器：

```bash
cd c:\digimon
python -m http.server 8000
```

然後在瀏覽器中訪問：`http://localhost:8000`

## 🎯 遊戲玩法

### 主選單
- **開始戰鬥** - 選擇數碼寶貝並開始對戰
- **數碼圖鑑** - 查看所有數碼寶貝的詳細資料
- **遊戲說明** - 了解遊戲規則和屬性相剋

### 戰鬥系統

#### 行動選項
1. **普通攻擊** - 基礎傷害，不消耗 MP
2. **技能攻擊** - 強力技能，消耗少量 MP
3. **必殺技** - 超強傷害，消耗大量 MP
4. **防禦** - 減少受到的傷害並恢復 10 MP

#### 屬性相剋
- 🔥 **火焰** → 克制 **自然** → 克制 💧 **水系** → 克制 🔥 **火焰**
- ✨ **光明** ↔ 🌑 **黑暗** (互相克制)
- 克制關係造成 **1.5 倍傷害**
- 被克制關係造成 **0.75 倍傷害**

## 🔧 自訂與擴展

### 添加新的數碼寶貝

編輯 `digimon-data.js` 檔案，在 `DIGIMON_DATABASE` 陣列中添加新的數碼寶貝物件：

```javascript
{
    id: 13,
    name: "你的數碼寶貝名稱",
    nameEn: "English Name",
    type: "fire", // fire, water, nature, light, dark
    typeLabel: "火焰",
    description: "描述文字",
    image: "圖片 URL",
    stats: {
        hp: 120,
        mp: 50,
        attack: 25,
        defense: 15,
        speed: 20
    },
    skills: [
        { name: "技能名稱", cost: 0, damage: 15, type: "normal" },
        // 更多技能...
    ]
}
```

### 調整遊戲平衡

在 `digimon-data.js` 中修改：
- `stats` - 調整數碼寶貝的能力值
- `skills` - 修改技能傷害和 MP 消耗
- `TYPE_EFFECTIVENESS` - 調整屬性相剋關係
